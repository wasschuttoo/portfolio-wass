# Wassil Chuttoo — Portfolio

A modern, premium personal portfolio built with Next.js 15, React 19, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel (recommended)

## Features

- Dark/light mode toggle with system preference detection
- Responsive design (mobile, tablet, desktop)
- Smooth scroll animations and micro-interactions
- SEO optimized with meta tags, Open Graph, Twitter Cards
- Schema.org structured data
- Sitemap and robots.txt
- Glassmorphism UI design
- Animated counters and floating elements
- Contact form (integrate with your preferred email service)
- Performance optimized (targets Lighthouse 95+)


## Deployment

Deploy to Vercel:

```bash
npx vercel
```

Or connect your GitHub repository to Vercel for automatic deployments.
